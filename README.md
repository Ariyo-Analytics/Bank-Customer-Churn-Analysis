# Customer Churn Analysis – Banking Sector
 
## 📌 Project Overview
This project analyzes customer churn drivers in a banking dataset using Power BI. It identifies high-risk customer segments that are likely to exit, enabling targeted retention strategies.
 
---
 
## 🔍 Key Insights
- Inactive customers have the highest churn rates.
- Customers aged 46–60 show elevated churn risk.
- Multi-product customers display significant attrition patterns.
 
---
 
## 🛠 Tools Used
- Power BI | DAX | SQL | Data Modeling | Data Visualization
